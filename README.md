# Cyber Playground


## User Stories - Cyber Playground

```
As a user so that I can make friends I would like to sign up to the website
```
```
As a user because I would like to be able to access my account I would like to login.
```
```
As a frustrated user that keeps losing I would like to be able to rage quit. 
```
```
As a lonely user I would like to be able to add people to my friends list. 
```
```
As a friendly person I would like to be able to communicate with all my friends online. 
```
```
As a user so that I can be entertained I would like to be able to play a game. 
```
```
As a user so I can ruin my friendships I would like to be able to play multiplayer games. 
```
```
As a user so that I can build my skills I would like to be able to join random games
```
```
As a collector of games I would like to be able to have a games list to view all of my games. 
```
